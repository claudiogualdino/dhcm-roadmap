Sincronizar Estados Jira

Os estados dos epics são obtidos via Claude + MCP Atlassian e guardados automaticamente no GitHub.
Como funciona o fluxo:
1 Clica "Abrir Claude" abaixo — abre uma nova conversa com a instrução pré-preenchida
2 Claude usa o MCP Atlassian para ir buscar o estado actual de cada epic + gerar briefings para os que ainda não têm
3 Claude atualiza o roadmap-state.json no GitHub com os estados obtidos
4 Volta ao dashboard e clica 🔄 Atualizar — os estados aparecem automaticamente
⚠️ Certifica-te que tens o MCP Atlassian ligado no Claude antes de avançar.
Settings → Integrations → Atlassian → Connect
